XML (eXtensible Markup Language) - это язык разметки, предназначенный для хранения и передачи данных. XML является
текстовым форматом и позволяет организовывать данные в структурированном виде с использованием тегов, атрибутов и
текстового содержимого.

XML-данные хранятся в виде древовидной структуры.

<person>
  <name>John Doe</name>
  <age>30</age>
  <email>johndoe@example.com</email>
  <isSubscribed>true</isSubscribed>
  <hobbies>
    <hobby>reading</hobby>
    <hobby>music</hobby>
    <hobby>sports</hobby>
  </hobbies>
  <address>
    <street>123 Main Street</street>
    <city>Anytown</city>
    <zipCode>12345</zipCode>
  </address>
</person>
