Автоупаковка - это автоматическая инкапсуляция примитивного типа в эквивалентную ему класс-обёртку всякий раз,
когда требуется объект данного типа.
Пример автоупаковки:
    Integer num = 9;

Автораспаковка - это преобразование класса-обёртки в соответствующий ему примитивный тип.
Пример автораспаковки:
    Integer num = 9;
    int x = num;
