ПО для обеспечения работы с изменяющейся информацией

СКВ дает возможность:
    - одновременно работать нескольким разработчиком над одной частью проекта;
    - видеть, кто, когда и какие изменения вносил;
    - позволяет формировать новые ветви проекта, объединять уже имеющиеся;
    - настраивать контроль доступа к проекту;
    - осуществлять откат до предыдущих версий.

Типы СКВ:
    - Локальные.
    на специально выделенном компе

    - Централизованные.
    Они имеют собственный центральный сервер, где накапливаются все файлы проекта (CVS, Subversion)

    - Распределенные.
    Данные хранятся в центральном репозитории, есть локальный репо на машине разработчика, например

Git - это ПО для подключения проекта к СКВ

GitHub - онлайн-хранилище кода
