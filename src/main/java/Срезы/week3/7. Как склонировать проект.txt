1. ЛР - УР +
    - копировать путь к репо (в гитлаб)
    - создать/открыть папку в которую будем клонировать и запустить GitBash
    - в командной строке GitBash, ввести команду git clone + скопированный path

2.  ЛР - УР -
    - создать УР (название репо пишется строчными англ буквами через дефис)
    - создать новый файл в репо или загрузить в УР файл с компа

    создание файла в репо через консоль GitBash:
    1. echo "project-name" >> README.md
    2. git init - Создать пустой репозиторий Git или вновь инициализировать существующий.
    При инициализации он создаст скрытую папку c расширением .git
    3. git add README.md
    4. git commit -m "first commit"
    5. git remote add origin + path к репо для установление связи с УР
    6. git push -u origin master

3. ЛР + УР -
    1. создать УР
    2. в GitBash переходим в ЛР
!   git status - показывает где мы есть
    3. git remote add origin + path к репо
    4. git push
! la-sa показывает имеющиеся в репо ЛР
! git remote -v показывает путь к УР который связан с ЛР
! git remote remove origin поменять путь к репо
